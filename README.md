# Игра змейка

## Как запустить

| Команда         | Описание                                     |
|-----------------|----------------------------------------------|
| `npm install`   | Установить зависимости                       |
| `npm run dev`   | Запустить локально на http://localhost:8080` |
| `npm run build` | Сделать сборку в папку `dist`                |
